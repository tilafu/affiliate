Usage Instructions

To use this clean header on other pages:

  1. Include the clean header loader script: <script src="./js/clean-header-loader.js"></script>
  2. Add a header placeholder: <div id="dashboard-header-placeholder"></div>
  3. Call: loadCleanStandardNavigation() in your page initialization
