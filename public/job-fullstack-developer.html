Job Overview

We're looking for a Senior Full-Stack Developer to join our engineering team and help build scalable web applications and APIs that power our global PEA network platform. You'll work on cutting-edge technology that connects thousands of Product Engagement Associates with e-commerce opportunities worldwide.

As a key member of our engineering team, you'll be responsible for developing and maintaining both frontend and backend systems, ensuring high performance, scalability, and user experience. You'll collaborate with product managers, designers, and other engineers to deliver features that impact millions of users globally.

Key Responsibilities

  • Design and develop scalable web applications using React, Node.js, and modern frameworks
  • Build and maintain RESTful APIs and microservices architecture
  • Optimize database performance and design efficient data models using PostgreSQL
  • Implement cloud infrastructure solutions using AWS services
  • Collaborate with cross-functional teams to define, design, and ship new features
  • Write clean, maintainable, and well-documented code
  • Conduct code reviews and mentor junior developers
  • Troubleshoot and debug applications to optimize performance
  • Stay up-to-date with emerging technologies and industry best practices

Required Qualifications

  • Bachelor's degree in Computer Science, Engineering, or related field
  • 5+ years of experience in full-stack web development
  • Strong proficiency in JavaScript, React, and Node.js
  • Experience with PostgreSQL or other relational databases
  • Hands-on experience with AWS cloud services
  • Proficiency in version control systems (Git)
  • Experience with CI/CD pipelines and DevOps practices
  • Strong problem-solving skills and attention to detail
  • Excellent communication and teamwork abilities

Preferred Qualifications

  • Experience with TypeScript and modern frontend frameworks
  • Knowledge of containerization technologies (Docker, Kubernetes)
  • Experience with message queues and event-driven architecture
  • Familiarity with testing frameworks (Jest, Cypress)
  • Experience in e-commerce or affiliate marketing platforms
  • Knowledge of data analytics and visualization tools

Technologies We Use

React Node.js TypeScript PostgreSQL AWS Docker Redis GraphQL Jest Cypress
